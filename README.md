## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/portfolio.git
```

### Navigate to project folder

```bash
cd portfolio
```

### Install dependencies

```bash
npm install
```

### Run locally

```bash
npm run dev
```

### Build project

```bash
npm run build
```
